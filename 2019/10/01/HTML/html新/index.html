1.1概要

网页主要分为：

​ W3C标准

  1. 结构 HTML
  2. 表现，样式 CSS
  3. 行为，响应用户操作 JavaScript

纯文本中只能保持文本内容，图片，音频，视频等格式化的内容，不能设置

HTML是超文本标记语言，纯文本是只有文字，不可以加图片，视频等格式的不可以

1.2常用标签及html骨架

<标签名></标签名>

根标签 头标签，不会在网页中直接显示，是用来帮助解析网页的 标题标签，会显示在浏览器的标题栏中，搜索引擎在检索页面时，

会首先检索title标签中的内容，是网页中对于搜索引擎来说

最重要的，会影响网页在搜索引擎中的排名

用来设置网页的主体内容，都应该在body中编写

可以通过属性来设置标签虾米处理标签中的内容，可以在开始标签中添加属性,属性需要写在开始标签中，实际上就是一个名/值对的结构 （x=1）这样的结构，一个标签中可以使用多个属性，但属性之间要使用空格隔开

color

size

doctype

为了让浏览器知道我们使用的是html的哪个版本，需要在网页的最上边添加一个doctype声明

如果不写文档声明，则会导致有些浏览器会进入一个怪模式，进入怪异模式之后，浏览器解析页面会导致页面无法正常显示，所以为了避免进入该模式，一定要写文档声明

怪异模式

为了兼容一些旧的页面，浏览器中设置了两种解析模式：

  1. 标准模式
  2. 怪异模式

怪异模式解析网页时会产生一些不可预期的行为，所以我们应该避免怪异模式的出现

避免的最好方式就是在页面中编写正常的doctype

center

可以设置居中，将代码放在它里面，建议用css，不用center

1.4常用的标签

<h1.> <h6.> 是六个标题标签

不关注标签的大小，因为关心的是语义，具体可以用css来定义大小什么的,<h1.>的重要性仅次于title，搜索引擎检索完title，会立即查看h1，页面只能写1个h1

段落标签p

<br/.>

在html中，字符之间写再多空格也相当于1个，在页面中可以用一个br标签来表示1个换行，br标签是一个自结束标签

hr标签

hr标签是一个自结束标签，可以在页面中生成一条水平线

1.5转义字符

在html中，一些如<>特殊字符是不能直接使用的,需要使用一些特殊字符，这些特殊符号就是转义字符

大于小于号： & 转义字符的名字;

​ (<) &lt.;

​ (>) &g.t;

空格: &nbs.p;

版权符号: &copy

1.6图片标签

使用img标签来向网页中引入一个外部图片

​ img标签也是一个自结束标签

属性：

​ src：设置一个外面图片的路径

​ alt: 可以用设置在图片不能显示时，对图片的描述

​ 搜索引擎可以通过alt属性来识别不同的图片

​ 如果不写alt属性，则搜索引擎不会对img中的图片进行收录

​ width:可以用来修改图片的宽度，一般使用px作为单位

​ height: 可以用来修改图片的高度，一般使用px作为单位

宽度和调试两个属性如果只设置1个，另一个也会同时等比例调整大小，如果两个值同时指定则按照指定的值来设置

一般开发中除了自适应的页面，不建议设置width和height

可以用../的方式在src下返回上一目录,返回几组目录，就写几个 ../

1.7图片的格式

JPG

  • JPG图片支持的颜色较多，图片可以压缩，但是不支持透明，一般使用jpg来保存照片等颜色丰富的图片

GIF

  • GIF支持的颜色多，并且支持复杂的透明
  • 可以用来显示颜色复杂的透明的图片

PNG

  • PNG支持的颜色多，并且支持复杂的透明
  • 可以用来显示颜色透明的图片

1.8meta标签

1.使用meta标签可以用来设置网页的关键字

2.还可以用来指定网页的描述，搜索引擎在检索页面时，会同时检索页面中的关键词和描述，但是这两个值 不会影响页面在引擎中的排名

3.使用meta可以用来做请求的重定向

1.9XHTML语法规范

  1. HTML中不区分大小写， 但是我们一般都使用小写
  2. HTML中的注释不能嵌套
  3. HTML标签必须结构完整，要么成对出现，要么自结束标签
  4. 浏览器尽最大的努力正确的解析页面，所有的不符合语法规范的内容，浏览器都会自动修正，但是有时会修正错误
  5. HTML标签可以嵌套，但是不能交叉嵌套
  6. HTML标签中的属性必须有值，且值必须加引号

使用内联框架可以引入一个外部的页面

2.1超链接

使用超链接可以让我们从一个页面跳转到另一个页面，使用a标签来创建一个超链接

属性:

​ href: 指向链接跳转的目标地址，可以写1个相对路径也可以写一个完整的地址

a标签中的target属性可以用来指定打开链接的位置，

​ 可选值:

  • _self:表示在当前窗口中打开(默认值)
  • _blank：在新的窗口中打开链接
  • 可以设置为一个内联框架的name属性值，链接将会在指定的内联框架中打开

创建超链接时，如果地址不确定可以直接写一个#作为占位符

如果将链接地址设置为#，则点击超链接后，会自动跳转到当前页面的顶部

去底部

先设置一个id，叫botton,之后在顶部查找botton,就可以实现快速到底部，每个元素都可以设置id，不过id头不可以以数字开头

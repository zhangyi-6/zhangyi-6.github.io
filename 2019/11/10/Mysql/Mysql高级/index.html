1.0.1 修改字符集

mysql> show variables like ‘%char%’;

会看到server的默认字符集是latin1(拉丁语)

cd /etc

ls -l my.cnf

vim my.cnf，加入：

service mysql stop ;

service mysql start;

只有搞完了这个后，新的数据库的字符集都是utf-8,老的数据库插入中文还是不行

1.0.2 索引

索引是一种数据结构 ，排好序的快速查找数据结构

数据本身之外，数据库还维护着一个满足特定查找算法的数据结构，这些数据结构眉高眼以某种方式指向数据，这样就可以在这些数据结构的基础上实现高级查找算法，这种数据结构就是索引

一般来说索引本身也很大，不可能全部存储在内存中，因此索引以文件的形式存储在磁盘上

优势：类似大学图书馆建书目索引，提高数据检索的效率，降低数据库的io成本

​ 通过索引列对数据进行排序，降低数据排序的成本，降低了cpu的消耗

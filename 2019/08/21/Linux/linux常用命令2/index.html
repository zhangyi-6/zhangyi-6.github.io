文件处理命令:touch

命令名称:touch
命令所在路径: /bin/touch
执行权限:所有用户
语法：touch[文件名]
功能：创建空文件
范例: 在当前文件夹下创建：
touch Japanlovestory.list
在别的文件夹下创建:
touch /root/Japanlovestory.list
创建多个文件中间用空格隔开:
touch a b c
创建带空格的文件
touch “program firles” ，用来表示是个整体

文件处理命令:cat

命令名称:cat
命令所在路径: /bin/cat
执行权限:所有用户
语法：cat[文件名]
功能：显示文件内容
-n 显示行号
范例: cat /etc/issue
cat -n /etc/services

值得一提的是将cat倒着写成tac，可以倒着显示文件内容
tac /etc/issue

文件处理命令:more

命令名称: more
命令所在路径： /bin/more
执行权限 :所有用户
语法： more [文件名]

  ( 空格）或者 f 翻页
（Enter)      换行
  q或Q         退出

能：分布显示文件内容
范例: more /etc/services

比more更好的是less可以通过PgUp向上翻页，通过向上的箭头，往上面的行查找

值得一提的是：
可以通过/（查询的名字）的方式查找
像我这里查找时输入/service ，会有高亮显示 ，按n键查找下一个service\

文件处理命令：ln 创建软链接（类似window快捷方式）

命令名称 ：ln

命令英文原意：link

命令所在路径 : /bin/ln

执行权限: 所有用户

语法: ln -s [原文件] [目标文件]

-s 创建软链接

功能描述：生成链接文件

范例：

软链接特征:类似windows快捷方式

1.lrwxrwxrwx ,软链接文件权限都为rwxrwxrwx

2.文件大小 只是符号链接

3./tmp/issue.soft -> /etc/issue 像这样，箭头指向源文件

硬链接特征：

  1. 相当于拷贝”cp - p” + 同步更新,源文件丢失，硬链接还可以继续使用，查找

    • ccho”www.lampbrother.net">>/etc/issue

  2. 通过i节点识别

  3. 不能跨分区 类似不能在windows上将C盘的东西copy到D盘 软链接可以的

  4. 不能针对目录使用 软链接是可以的

权限管理命令chmod

命令英文原意:change the permissions mode of a file

命令所在路径 : /bin/chmod

执行权限: 所有用户

权限的数字表示

r—–4

w—-2

x—–1

rwxrw-r–

7 6 4

范例:

chmod g + w testfile

赋予文件testfile所属组写权限

chmod -R777 testdir

修改目录testfile及其目录下文件为所有用户具有全部权限

文件目录权限总结

代表字符 权限 对文件的含义 对目录的含义 对于文件来说可以用的命令 对于目录来说可以用的命令
r 读权限 可以查看文件内容 可以列出目录中的内容 cat/more/head/tail/less ls
w 写权限 可以修改文件内容 可以在目录中创建、删除文件 vim touch/mkdir/rmdir/rm
x 执行权限 可以执行文件 可以进入目录 script command cd

权限管理命令:chown

命令名称:chown

命令英文原意:change file ownership

命令所在路径： /bin/chown

执行权限：所有用户

语法:chown [用户] [文件或目录]

功能描述:改变文件或目录的所有者，只可以是root操作，别的不可以

范例 : chown shenchao fengjie

改变文件fengjie的所有者为shenchao

范例： chgrp lampbrother fengjie

改变文件fengjie的所属组为lampbroter

文件搜索命令：find

命令名称: find

命令所在路径： /bin/find

执行权限：所有用户

语法： find [搜索范围] [匹配条件]

功能描述：文件搜索

例：

find /etc -name init

  • 在目录/etc中查找文件init，注意:在init的两边加上”*”,就像通配符一样，查找，不然就只是找init，而不是包含init

  • init* 表示查找以init开头的文件

  • init???代表以init开头的，后面带3个字母的

    • -name不区分大小写

find / -size +204800

  • 在根目录下查找大于100MB的文件
    • +n 大于 -n小于 n等于
  • 1个数据块 512字节 0.5k 所以1k等于2个数据块
    • 100MB =102400KB=204800

find/ home -user shenchao

  • 在根目录下查找所有者为shenchao的文件
    • -group 根据所属组查找

根据时间属性来查找

find /etc -cmin -5

  • 在/etc下查找5分钟内被修改过属性的文件和目录

    • -amin 访问时间 access

    • -cmin 文件属性 change

    • -mmin 文件内容 modify

  • find /etc-size +163840 -a -size -204800

    在/etc下查找大于80MB小于100MB的文件

    -a两个条件同时满足

    -o两个条件满足任意一个即可

  • find /etc -name inittab -exec ls -l｛};

在/etc下查找inittab文件并显示其详细信息

-exec/-ok 命令{} ; 对搜索结果执行操作

-type 根据文件类型查找

f 文件 d目录 l软链接文件

-inum 根据i节点查找

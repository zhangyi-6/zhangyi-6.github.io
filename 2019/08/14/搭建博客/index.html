关于用hexo搭建框架

先下载nodejs

https://nodejs.org/en/

cmd管理员模式下：

  1. node -v(查看node版本)
  2. npm -v(查看mpm安装工具版本)

用淘宝的镜像源cnpm 原本的npm太慢

npm install -g cnpm --registry=https://registry.npm.taobao.org

现在文件已经到了:

C:\nodejs\node_global\node_modules\cnpm\bin\cnpm 这里了

输入:

cnpm  验证cnpm是否安装成功
用cnpm安装 hexo:
cnpm install -g hexo-cli

像下面这样:

用hexo -v 验证:

先 cd C:\nodejs
mkdir blog ，如果在WINDOWS\system32下 mkdir blog，就会跑到system32下了

hexo init（初始化博客）

成功了

可能出现的问题

cnpm不是内部文件，这种一般是环境变量没配

之后在环境变量中配置下就好了：

error deployer not found:github的解决方法
2017年01月30日 14:08:03 acm_cxq 阅读数 509
版权声明：本文为博主原创文章，遵循 CC 4.0 by-sa 版权协议，转载请附上原文出处链接和本声明。
本文链接：https://blog.csdn.net/acm_cxq/article/details/54782911

在部署hexo的时候遇到过几次这个问题，每次都是用这个方法解决的。

先注意type冒号之后是不是有记得加了空格

如果是，就运行一下 $npm install hexo-deployer-git –save

然后再 hexo deploy

问题：hexo配置好后，运行hexo clean && hexo g && hexo d命令，出现ERROR Deployer not found: git

原因：少了hexo针对git的deploy组件

解决方法：npm install –save hexo-deployer-git

搞定，运行成功之后提示：INFO Deploy done: git

安装next主题：
在git下输入:
Git clone https://github.com/theme-next/hexo-theme-next themes/next
下载好后，找到根目录下的_config.yml 将里面的theme改成:
theme: next

编码：依据一定的规则，将字符转换为二进制编码的过程

解码：依据一定的规则，将二进制编码转换为字符的过程

字符集：编码和解码所采用的规则，我们称为字符集

常见字符集：

  1. ASCII 7位，128个字符集 美国人 发明的
  2. ISO-8859 8位 256字符集 欧洲发明
  3. GBK GB意为国标 中国发明
  4. GB2312 中文系统的默认编码
  5. UTF-8 万国友，支持所有的文字
  6. ANSI 自动以系统的默认编码保存文件，记事本就是这么保存html页面的，系统是什么解码，它就以什么编码

产生乱码的根本原因是，编码和解码采用的字符集不同

在中文系统的浏览器中，默认都是GB2312

这时，就需要告诉网页我们的编码字符集：

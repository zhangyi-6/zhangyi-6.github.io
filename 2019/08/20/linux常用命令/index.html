linux常用命令

发表于 2019-08-20 更新于 2019-08-21

clear清除屏幕

命令格式

命令 [-选项] [参数]
例: ls -la etc

  1. 个别命令使用不遵循此格式
  2. 当夺多个选项时，可以写在一起
  3. 简化选项与完整选项 ls -a等于ls - all （查看所有）

#ls#
英文原意:list
命令所在路径: /bin/ls
执行权限：所有用户
功能描述:用户目录文件
语法 ：ls 选项[-ald] [文件或目录]

  • 显示所有文件，包括隐藏文件
  • 详细信息显示
  • 查看目录属性

ls -l命令

ls -l (长格式显示)

说一下上面的倒数第二行的语句

-rw——- .1 root root 1438 8月 20 13:24 anaconda-ks.cfg
-rw-r–r–
-r读 w写 x执行

  • u g o
  • u所有者 g所属组 o其他人
  • r读 w写 x执行
  • 1代表引用计数
  • root代表所有者，所有者只能有一个
  • 第二个root代表所属组，定义了一组同类型的用户，也是只能有一个
  • 1438代理文件大小，下面有给出人性化查看文件大小的方法，ls -lh
  • 下面是文件最后修改时间
  • 最后是文件名

最常见的3种文件类型

  • 前面是-表示这是个文件
  • 前面是d表示这是个目录
  • 前面是l表示这是个软链接

ls -lh

ls -lh(人性化显示human)

当我们想查看某个文件夹的本身属性时

ls -ld /（某个文件夹）

ls -i

ls -i 可以查看当前文件夹的节点

目录处理命令:mkdir

  • 英文原意：make directories
  • /命令所在路径：/bin/mkdir
  • 执行权限:所有用户
  • 语法:mkdir -p 目录名
  • 功能:

1.创建新目录
2.-p 递归创建
mkdir后加上-p,可以在本身不存在的文件夹下创建文件
mkdir -p /tmp/Japan/boduo

同时创建多个文件:
mkdir /tmp/Japan/longze /tmp/Japan/cangjing

目录处理命令 pwd

命令名称:pwd
命令英文原意:print working directory
执行权限:所有用户
语法:pwd
功能描述：显示当前目录
范例:直接输入:
pwd

文件处理命令

命令名称:rmdir
命令英文原意:remove empty directories
命令所在路径 ：/bin/rmdir
执行权限:所有用户
语法 ：rmdir[目录名]
功能: 删除空目录
范例: rmdir /tmp/Japan/boduo

目录处理命令 cp

命令名称:cp
命令英文原意：copy
命令所在路径 : /bin/cp
执行权限: 所有用户
语法 : cp -rp [原文件或目录][目标目录]
-r 复制目录
-p 保留文件属性
功能:
复制文件或目录

如果只是复制文件，就不用写 -r或-rp了

可以同时复制多个文件:
cp /root/install.log /root/install.log.syslog /tmp
下面是添加了-p后的命令，加了-p可以连文件的属性，比如原文件夹的创建时间
cp -p /root/install.log /root/install.log.syslog /tmp

还可以再复制文件夹时改名：
cp -r /tmp/Japan/longzeluola /root/xixi.133+

目录处理命令: mv

命令名称:mv
命令英文原意：move
命令所在路径 : /bin/mv
执行权限: mv [原文件或目录] [目标目录]
功能: 剪切文件、改名

改文件夹的名字 mv () ()

mv cangjingkong canglaoshi

目录处理命令:rm

命令名称: rm
命令英文原意: remove
命令所在路径 ：/bin/rm
执行权限: 所有用户
语法: rm -rf [文件或目录]
-r 删除目录
-f 强制执行
功能: 删除文件

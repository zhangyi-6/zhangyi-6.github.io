    java语言的特性

跨平台性

通过java语言编写的应用程序在不同的系统平台上都可以运行

原理

JVM虚拟机，JVM是要下载对应的版本的，比如从windows上下载的东西要放到linux上，需要在linux上下载对应的jdk

搭建环境

  • JRE(java运行环境) 包括JVM和java程序所需的核心类库，想运行java程序，需要安装

  • JDK(java开发工具包) 供java开发人员使用，里面包括JRE，编译工具(javac.exe)打包工具(jar.exe)等

    JDK下载地址:https://www.oracle.com/technetwork/java/javaee/downloads/index.html

    jdk8API文档地址:https://javaee.github.io/javaee-spec/javadocs/

标识符

  • 在程序中自定义的一些名称。

  • 由26个英文字母大小写，数字：0-9符号：_$组成

  • 定义合法标识符规则：

    1. 数字不可以开头
    2. 不可以使用关键字

  • Java中严格区分大小写

java中的名称规范：

  • 包名：多单词组成时所有字母都小写: xxxxxxyyyyyzzz

  • 类名接口名：多单词组成时，所有的单词的首字母大写：XxxYyyZzz

  • 变量名和函数名：多单词组成时，第一单词首字母小写，第二个单词开始每个单词首字母大写。

    xxxYyyZzz

  • 常量名：所有字母都大写，多单词时每个单词用下划线连接。

    XXX_YYYX_ZZZ

常量与变量

java中常量的分类：

  1. 整数常量，所有整数
  2. 小数，所有小数
  3. 布尔常量，true,false
  4. 字符常量，’ ‘单引号表示
  5. 字符串常量，” “ 双引号表示
  6. null常量，只有一个null
  • 对于整数，java有三种表现形式：
  • 十进制：0-9 满10进1
  • 八进制: 0-7 满8进1 用0开头 3位代表1位
  • 十六进制: 0-9，A-F，满16进1，用0x开头 4位代表1位

进制越大表现形式越短

逻辑运算符
& 与 false&true false
| 或 ^ 异或 false|true true^false true true
！非 &&短路 ！true false&&true false false
||短路 false||true true

&:只要两边的boolean表达式一个是false，那么结果就是false,只有两边都是true，结果为true

| 两边只要有一个true，结果为true，只有两边都有false，结果为false

^ 异或 true^ true =false; 两边结果相同结果是false

​ 两边不同结果是true
